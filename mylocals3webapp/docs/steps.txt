Login to AWS Console https://ap-south-1.console.aws.amazon.com/

Step 1: Create a S3 Bucket 
Step 2: enable Static Website Hosting on your bucket
Step 3: Create a Git Repository 
Step 4: Create an IAM User, enable Access key and Secret Key for the IAM User
Step 5: Add Secrets to the repository --> Go to Settings-->Secrets and Variable--->Actions---> Add repository Secrets
Note: Make sure to add your bucket Id , Access key and Secret Key
Step 6: Install GitHub on your local system, initial the repository and commit changes to it
